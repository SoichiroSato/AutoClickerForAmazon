＃AutoClickerForAmazon
アマゾンで自動購入するツールです。
ログインIDとログインpasswordと購入したい商品のURLと購入時刻を入力して
指定された時刻にSeleniumによる自動操作で購入させます。
対応ページは指定時刻に「今すぐ購入」ボタンが押せるページのみになります。
「注文内容を確認・変更する」画面で「注文を確定する」ボタンが表示されることが条件になります。
